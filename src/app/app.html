Que es

Un laboratorio remoto moderno para ensenar control automatico

Permite realizar practicas reales sin depender por completo de un laboratorio fisico tradicional. El alumno entra desde el navegador, interactua con el sistema y observa resultados en graficas.

Acceso desde cualquier lugar

Facilita clases remotas, hibridas y practicas fuera del horario del laboratorio.

Hardware de bajo costo

Usa tecnologias accesibles como ESP32 para reducir costos de implementacion.

Practicas

Compara diferentes estrategias de control

Los usuarios pueden probar metodos sobre el mismo sistema y observar diferencias de comportamiento.

Control PI

Logica difusa

Experimenta con reglas difusas para analizar sistemas no lineales.

Red neuronal

Aplica inteligencia artificial y compara resultados con otros metodos.
